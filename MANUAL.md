## Админ-панель

### Вход в админ-панель

1. Откройте `/login`
2. Введите: `admin@buhgalter.tech` / `admin123`
3. Нажмите "Войти"
4. Вы будете перенаправлены на `/admin`

### Разделы админ-панели

#### 1. Дашборд (`/admin`)
- Общая статистика: лиды, клиенты, доход
- Графики конверсии
- Последние заявки

#### 2. Управление услугами (`/admin/services`)

**Добавление услуги:**
1. Нажмите "Добавить услугу"
2. Заполните поля:
   - Название
   - Описание
   - Цена (например: "от 10 000")
   - Примечание к цене (например: "руб/мес")
   - Иконка (выберите из списка)
   - URL страницы
3. Нажмите "Сохранить"

**Редактирование услуги:**
1. Найдите услугу в списке
2. Нажмите иконку редактирования (карандаш)
3. Измените нужные поля
4. Нажмите "Сохранить"

**Изменение порядка:**
- Используйте поле "Порядок" для сортировки услуг на главной странице

#### 3. Управление партнёрами (`/admin/partners`)

**Добавление партнёра:**
1. Нажмите "Добавить партнёра"
2. Заполните:
   - Название (например: "Сбер Банк")
   - Описание
   - Цвет бренда (HEX, например: #21A038)
   - URL сайта
   - Бонус (например: "+ 500 руб бонус")
   - Возможности (список через Enter)
   - Наши услуги (список через Enter)
3. Нажмите "Сохранить"

#### 4. Управление отзывами (`/admin/testimonials`)

**Добавление отзыва:**
1. Нажмите "Добавить отзыв"
2. Заполните:
   - Имя клиента
   - Компания
   - Должность
   - URL аватара (или оставьте пустым)
   - Рейтинг (1-5 звёзд)
   - Текст отзыва
   - Дата (например: "Март 2026")
3. Установите "Опубликован" для показа на сайте
4. Нажмите "Сохранить"

#### 5. CRM и аналитика (`/admin/crm`)

**Просмотр лидов:**
- Все заявки с сайта отображаются в таблице
- Фильтрация по статусу: Новый, В работе, Завершён, Отклонён
- Поиск по имени, телефону, email

**Изменение статуса лида:**
1. Найдите лид в списке
2. Нажмите на статус
3. Выберите новый статус из выпадающего списка

**Экспорт данных:**
- Нажмите "Экспорт в CSV" для выгрузки всех лидов

#### 6. Управление блогом (`/admin/blog`)

**Создание статьи:**
1. Нажмите "Новая статья"
2. Заполните:
   - Заголовок
   - Slug (URL статьи, например: `kak-vybrat-sistemu-nalogooblozheniya`)
   - Краткое описание
   - Содержание (поддерживает Markdown)
   - Категория
   - Теги (через запятую)
   - Изображение (URL)
3. Установите "Опубликовать" если готово
4. Нажмите "Сохранить"

### Вкладки личного кабинета

- Чат с персональным бухгалтером
- Отправка текстовых сообщений

1. Нажмите иконку скрепки (📎) рядом с полем ввода
2. Выберите файл на компьютере
3. Файл автоматически загрузится и отправится

- Список всех документов по компании

- Статусы документов:
  - 🟡 На проверке
  - 🟢 Готов
  - 🔴 Требует внимания
- Скачивание документов

#### 3. Задачи
- Текущие задачи по бухгалтерии
- Сроки выполнения
- Статусы: Новая, В работе, Завершена

#### 4. Настройки
- Изменение пароля
- Настройка уведомлений

## Калькулятор стоимости

### Расположение
- Главная страница (виджет)
- Отдельная страница `/calculator`

### Как пользоваться

1. **Выберите форму бизнеса:**
   - ИП
   - ООО

2. **Выберите систему налогообложения:**
   - УСН (для ИП и ООО)
   - Патент (только для ИП)
   - УСН + НДС (для ИП и ООО)
   - ОСН (только для ООО)

3. **Укажите наличие оборота:**
   - Нулевой оборот (нет деятельности)
   - Есть оборот (активная деятельность)

4. **Укажите сотрудников (если есть оборот):**
   - Без сотрудников
   - 1-3 сотрудника

5. **Результат:**
   - Стоимость в месяц
   - Что входит в стоимость
   - Кнопки "Оставить заявку" и "Позвонить"

### Таблица тарифов

| Форма | Система | Без оборота | С оборотом | С сотрудниками |
|-------|---------|-------------|------------|----------------|
| ИП | УСН/Патент | 5 000 ₽ | от 10 000 ₽ | от 15 000 ₽ |
| ИП | УСН+НДС | 5 000 ₽ | от 15 000 ₽ | от 20 000 ₽ |
| ООО | УСН | 10 000 ₽ | от 15 000 ₽ | от 25 000 ₽ |
| ООО | УСН+НДС/ОСН | 10 000 ₽ | от 20 000 ₽ | от 25 000 ₽ |

## Форма обратной связи

### Расположение
- Главная страница (секция "Закажите надёжную бухгалтерию")
- Страница контактов `/contacts`

### Поля формы
- **Имя** (обязательно)
- **Телефон** (обязательно, формат: +7 999 999 99 99)
- **Вопрос** (необязательно)

### Что происходит после отправки
1. Заявка сохраняется в CRM
2. Отображается в админ-панели (`/admin/crm`)
3. Статус: "Новый"
4. Показывается уведомление об успешной отправке

### Валидация
- Имя: минимум 2 символа
- Телефон: валидный российский номер
- Защита от спама: проверка на частые отправки

## Push-уведомления

### Включение уведомлений (для клиентов)

1. Войдите в личный кабинет
2. Перейдите во вкладку "Настройки"
3. Включите переключатель "Push-уведомления"
4. Разрешите уведомления в браузере

### Типы уведомлений

| Тип | Описание | Можно отключить |
|-----|----------|-----------------|
| Документы | Новый документ загружен | Да |
| Сообщения | Новое сообщение от бухгалтера | Да |
| Напоминания | Сроки оплаты и сдачи отчётности | Да |
| Статусы | Изменение статуса документа | Да |
| Промо | Акции и спецпредложения | Да |

### Тихие часы
- Можно настроить время, когда уведомления не приходят
- По умолчанию: 22:00 - 08:00

## Техническая информация

### Стек технологий
- **Frontend:** Next.js 15, React 18, TypeScript
- **Стили:** Tailwind CSS, shadcn/ui
- **Хостинг:** Netlify
- **Иконки:** Lucide React

### Хранение данных
Все данные хранятся в localStorage браузера (демо-режим):
- `buhgalter_services` - услуги
- `buhgalter_partners` - партнёры
- `buhgalter_testimonials` - отзывы
- `buhgalter_contacts` - контакты
- `buhgalter_leads` - заявки
- `buhgalter_users` - пользователи
- `buhgalter_blog_posts` - статьи блога

### Сброс данных
Для сброса всех данных к значениям по умолчанию:
1. Откройте консоль разработчика (F12)
2. Введите: `localStorage.clear()`
3. Обновите страницу

## Часто задаваемые вопросы

### Как изменить контактные данные на сайте?

1. Войдите в админ-панель
2. Перейдите в "Настройки" → "Контакты"
3. Измените телефон, email, адрес
4. Нажмите "Сохранить"

### Как добавить нового бухгалтера?

В демо-версии невозможно. В продакшн-версии:
1. Админ-панель → Пользователи
2. "Добавить пользователя"
3. Выберите роль "Бухгалтер"

### Как подключить реальную CRM?

Сайт поддерживает интеграцию с:
- AmoCRM
- Bitrix24
- Webhook (для любой CRM)

Настройка в файле `/src/lib/crm.ts`.

### Как добавить Яндекс Метрику?

1. Получите код счётчика в Яндекс Метрике
2. Добавьте ID счётчика в файл `/src/lib/ab-testing.ts`
3. Раскомментируйте код аналитики

### Как изменить цвета сайта?

1. Откройте `/src/app/globals.css`
2. Найдите секцию `:root`
3. Измените CSS-переменные:
   - `--primary` - основной синий цвет
   - `--secondary` - красный акцентный цвет
